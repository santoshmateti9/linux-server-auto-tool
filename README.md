## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/linux-server-auto-tool.git
```

Move into the project:

```bash
cd linux-server-auto-tool
```

Give execute permission to a script:

```bash
chmod +x scripts/system_info.sh
```

Run the script:

```bash
./scripts/system_info.sh
```

or

```bash
bash scripts/system_info.sh
```
